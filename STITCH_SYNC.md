# Sync theme from Google Stitch

Stitch project: https://stitch.withgoogle.com/projects/11195237751914514939

The app theme lives in `src/styles/stitch-theme.css`. To match your Stitch design exactly:

1. Open the project in Stitch (while signed in).
2. Select a screen → **Code** or **⋯ → Export** → copy CSS / `DESIGN.md`.
3. Paste color and font values into the `:root` block in `stitch-theme.css`.

Or enable **Share Project → Public prototype** and send the preview link so tokens can be pulled from the live UI.

## What’s applied now (M3 / Stitch-style)

- **Fonts:** Fraunces (display) + Inter (UI)
- **Colors:** Forest green primary, warm cream surfaces, ghee-gold tertiary
- **Shape:** 16–28px rounded cards, pill buttons
- **Components:** M3 tabs, elevated cards, tonal bottom nav active state
